# 성적분석프로그램 사용 가이드

## 빠른 시작

### 1. 프로그램 실행

```bash
streamlit run app.py
```

브라우저에서 자동으로 http://localhost:8501 열림

### 2. 파일 준비

#### 정기고사 분석
1. NEIS에서 **문항정보표** 다운로드 (엑셀)
2. NEIS에서 **학생답정오표** 다운로드 (학급별로 여러 개 가능)
3. NEIS에서 **성적일람표** 다운로드 (학기말 성취도 기반 시)

#### 수행평가 분석
1. **평가기준표** 준비
2. **성적일람표** 다운로드

### 3. 분석 설정

1. **분석 기준 선택**
   - 분할점수 기반: A/B 90점, B/C 80점 등 입력
   - 학기말 성취도 기반: 성적일람표의 성취도 사용

2. **평가 계획 설정**
   - 평가명, 만점, 반영비율 입력
   - 정기고사/수행평가 구분

3. **파일 업로드**
   - 사이드바에서 필요한 파일 선택

4. **결과 확인**
   - 6개 탭에서 다양한 분석 결과 확인

## 파일 형식 주의사항

### 문항정보표
- NEIS 표준 양식 사용
- 10행부터 데이터 시작
- 문항 번호가 첫 번째 열에 있어야 함

### 학생답정오표
- 문항 번호(1, 2, 3, 4...)가 헤더에 있어야 함
- 학생 이름이 있어야 함
- 반/번호 형식: "1/1", "2-15" 등

### 성적일람표
- "성명" 또는 "이름" 열 필요
- "성취도" 또는 "등급" 열 필요
- 반/번호 열 권장

## 문제 해결

### Q1. 파일 업로드 후 오류 발생
**A:** 파일 형식이 NEIS 표준과 다를 수 있습니다. 다운로드 옵션을 확인하세요.

### Q2. 학생 수 불일치 경고
**A:** 정오표와 성적일람표의 학생 이름이 다릅니다. 전출/자퇴 학생이 있는지 확인하세요.

### Q3. 성취도 판정이 이상함
**A:** 분할점수를 확인하세요. 또는 학기말 성취도 기반으로 변경하세요.

### Q4. 그래프가 표시되지 않음
**A:** 브라우저를 새로고침(F5)하세요. 캐시를 지워보세요.

### Q5. 테이블 정렬이 안 됨
**A:** 헤더를 클릭하여 정렬할 수 있습니다. DataTables 기능을 사용합니다.

## 기능별 가이드

### 데이터 탭
- 문항정보표 확인
- 학생 성적 데이터 확인
- 점수 분포 그래프

### 전체 성취도 분석 탭
- 강의실별 필터링
- 성취수준별 학생 수
- 평균 점수 비교

### 문항 분석 탭
- 선택형 문항 정답률/변별도
- 답지반응 분포
- 성취수준별 정답률
- P-D Chart

### 답지반응 분포 탭
- 문항별 선택
- 성취수준별 정답률 추이

### 성취기준 분석 탭
- 성취기준별 통계
- 평균 정답률/변별도

### AI 리포트 탭
- 신뢰도 진단
- 우수 문항 식별
- 개선 제언

## 고급 설정

### 성취수준 구분
- 3수준: A, B, C
- 5수준: A, B, C, D, E
- 5수준+미도달: A, B, C, D, E, 미도달

### 분할점수 예시
- **80% 기준**: A/B 80점, B/C 60점
- **표준**: A/B 90점, B/C 80점, C/D 70점, D/E 60점
- **미도달 포함**: E/미도달 40점 추가
